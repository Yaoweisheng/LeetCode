# [466. 统计重复个数](https://leetcode-cn.com/problems/count-the-repetitions/)

> 解题思路分析

- 暴力解居然能过很惊讶
- 寻找单次循环，找到loopNum1个s1可以转换成loopNum2个s2，求得n1/loopNum1*loopNum2，然后再将无法整除的部分n1%loopNum1顺序遍历加到结果中。
- 查找单次循环，记录第一次s2的索引对应的s1的索引，当第二次出现这个索引对时，即说明循环了一遍了。
- 需要通过记录第一次s2末尾的索引对应的s1的索引，保证一次循环完成，通过除了末尾的索引记录可能会出现其实到这里就结束了，但多记录了一次。


### 代码实现


~~~java
// package com.yws.AC;

import java.util.HashMap;
import java.util.Map;

class Solution {
    public int getMaxRepetitions(String s1, int n1, String s2, int n2) {
        char[] cs1 = s1.toCharArray();
        char[] cs2 = s2.toCharArray();
        int index1 = -1;
        int index2 = 0;
        int prei = 0;
        int prej = 0;
        int loopNum1 = 0;
        int loopNum2 = 0;
        int ans = 0;
        boolean flag = false;
        for(int i = 0; i < n1; i++){
            if(flag){
                break;
            }
            for(int j = 0; j < cs1.length; j++){
                if(cs1[j] == cs2[index2]){
                    if(index2 == cs1.length-1){
                        if(index1 == -1){
                            index1 = j;
                            prei = i;
                            prej = ans;
                        } else{
                            if(index1 == j){
                                flag = true;
                                loopNum1 = i-prei;
                                loopNum2 = ans-prej;
                                break;
                            }
                        }
                    }
                    index2++;
                    if(index2 == cs2.length){
                        ans++;
                        index2 = 0;
                    }
                }
            }
        }
        if(flag){
            ans = prej + (n1-prei)/loopNum1*loopNum2;
            int rest = (n1 - prei)%loopNum1;
            for(int i = 0; i < rest; i++){
                for(int j = 0; j < cs1.length; j++){
                    if(cs1[j] == cs2[index2]){
                        index2++;
                        if(index2 == cs2.length){
                            ans++;
                            index2 = 0;
                        }
                    }
                }
            }
        }
        return ans/n2;
    }
//

    //     System.out.println(maxRepetitions);
    // }
}
~~~
